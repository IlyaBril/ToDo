# README

## О приложении

Редактируемая таблица (ToDo List) на Gridjs, Flask

## Запустить приложение


### Поднять с помощью docker

Собрать образ: docker build -t mytodo .

Запустить: docker run -d -p 80:80 -p 81:81 mytodo

В браузере localhost:80
